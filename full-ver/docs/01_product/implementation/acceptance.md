## Authentication

- [ ] User can log in with valid email and password
- [ ] User sees error message on invalid credentials
- [ ] User is redirected to home after login
- [ ] User can log out and session is cleared
- [ ] JWT token is invalidated after logout

---

## User Management (Admin)

- [ ] Admin can list all users
- [ ] Admin can create a new user with email and role
- [ ] Admin can change a user's role
- [ ] Admin can deactivate a user (soft delete)
- [ ] Admin cannot deactivate the last admin account

---

## Profile

- [ ] User can view their own profile
- [ ] User can update their display name
- [ ] User can change their password with current password confirmation
