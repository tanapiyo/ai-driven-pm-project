# H01. ホーム（ロール別ダッシュボード）

**画面ID**: H01
**画面名**: ホーム
**対象ロール**: 全ロール（認証後）

---

## 目的

ログイン後の初期画面。ロール別にパーソナライズされたショートカット・サマリー・推奨アクションを表示。

---

## 画面仕様（ロール別出し分け）

### ヘッダ（全ロール共通）

- **グローバルナビゲーション**: メニュー（権限で表示制御）
- **ユーザーメニー**: プロフィール, ログアウト

---

## Admin 向けセクション

### セクション構成

| セクション | 説明 | 表示項目 |
|----------|------|--------|
| システム健全性 | データ品質チェック | 担当未設定数, 欠損数（リスク別） |
| 最近のユーザー操作 | 監査ログ（抜粋） | Last 20 操作（ユーザー、操作種別、エンティティ、タイムスタンプ） |
| ユーザー管理 | アカウント状況 | アクティブユーザー数, 今月新規 |
| クイックアクション | 管理機能へ | 「ユーザー管理（AD01へ）」「マスタ管理（AD02へ）」「監査ログ確認（AD03へ）」 |

---

## 共通仕様

### レスポンシブ対応

- **デスクトップ**: 2〜3 カラムレイアウト
- **タブレット**: 2 カラムレイアウト
- **モバイル**: 1 カラムレイアウト（スタック）

### リフレッシュ

- 初回ロード: API から全データ取得
- 再訪問時: キャッシュあれば使用、15 分以上古ければ再取得

---

## 受入条件

- [ ] ロール別にセクションが正しく出し分けられる
- [ ] 各セクションのサマリーが正確に集計される
- [ ] クイックアクションボタンが正しい画面に遷移する
- [ ] ロード時間が 3 秒以下
- [ ] ユーザー操作の権限チェックが適用される
