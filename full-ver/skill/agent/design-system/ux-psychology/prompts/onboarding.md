# UX Psychology Onboarding Prompt

新メンバーやプロジェクト参加時に UX心理学パックの概要を説明するためのプロンプト。

---

## Usage

チームへのオンボーディング時に使用する。

## Prompt

```
UX心理学パックについて説明してください。

## パックの概要
skill/agent/design-system/ux-psychology/ に配置されたツールとテンプレートのセットです。
UX心理学の概念をプロダクト設計・AC定義・PRレビュー・学習サイクルに統合します。

## 説明してほしいこと
1. このパックの目的と価値
2. 開発フローの中でどこで使うか（設計→AC→レビュー→学習）
3. 最低限知っておくべきテンプレートとその使い方
4. ダークパターン防止の重要性と 3つの自問
5. よくある質問（必要最小限でよい、全概念を覚える必要はない、等）

## 対象者の前提知識
- UX心理学の知識: [初心者 / 中級 / 上級]
- 開発プロセスの理解: [新規 / 既存メンバー]

## Output
5分で読める簡潔な説明にしてください。
```
